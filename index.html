Personal Project

Real-Time 3D Tactical Map — MCP / WebSocket Architecture

TypeScript React React Three Fiber WebSocket Zustand Zod pnpm workspaces

Real-time 3D multiplayer tactical map with a WebSocket-based authoritative server, a REST-to-intent bridge agent, and a React Three Fiber UI. The server maintains a validated 3D voxel grid (terrain, fog-of-war, tokens, obstacles, effects) and broadcasts diffs to all connected clients. A world-ops HTTP agent bridges REST requests to typed MCP intents with stepwise pathfinding and a job queue. Four packages in a pnpm monorepo share Zod protocol schemas, error codes, and encode/decode helpers.

Protocol & State

Intent → validate → ACK + PATCH broadcast model. Authoritative server owns all mutations; clients apply patches optimistically with server reconciliation.

Agent Bridge

HTTP → MCP intent bridge with stepwise pathfinding, token CRUD, terrain/fog/effect mutation, save/load, and a job queue for multi-step operations.

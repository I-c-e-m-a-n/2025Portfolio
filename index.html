Accessible Interactive Music System

C++ C# Unity Python Arduino Android Bluetooth Generative AI

Built as part of the University of Delaware's Sensify Lab VIP program (2022). The system makes music participation accessible to children on the autism spectrum by letting listeners activate, disable, and recombine musical layers (melody, rhythm, harmony) according to their sensory preferences. Interaction & Logging

Layer toggle events captured with timestamps to CSV. Session replay and visualization tooling in Python for behavioral analysis across pilot sessions.

Stack

Unity and C# for runtime orchestration, Arduino and Bluetooth for hardware input, Android plugin for mobile deployment, Python for AI integration and log analysis.

Personal Project

Real-Time 3D Tactical Map — MCP / WebSocket Architecture

TypeScript React React Three Fiber WebSocket Zustand Zod pnpm workspaces

Real-time 3D multiplayer tactical map with a WebSocket-based authoritative server, a REST-to-intent bridge agent, and a React Three Fiber UI. The server maintains a validated 3D voxel grid covering terrain, fog-of-war, tokens, obstacles, and effects, and broadcasts diffs to all connected clients. An HTTP world-ops agent bridges REST requests to typed MCP intents with stepwise pathfinding and a job queue. Four packages in a pnpm monorepo share Zod protocol schemas, error codes, and encode/decode helpers.

Protocol & State

Intent, validate, ACK + PATCH broadcast model. Authoritative server owns all mutations; clients apply patches optimistically with server reconciliation.

Agent Bridge

HTTP to MCP intent bridge with stepwise pathfinding, token CRUD, terrain/fog/effect mutation, save/load, and a job queue for multi-step operations.
